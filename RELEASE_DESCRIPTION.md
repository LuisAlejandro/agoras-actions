*Agoras Actions* is a GitHub Action that wraps [Agoras](https://github.com/LuisAlejandro/agoras) for publishing, scheduling, liking, sharing, and deleting posts on social networks. It supports X, Facebook, Instagram, LinkedIn, Discord, YouTube, TikTok, Threads, Telegram, and WhatsApp. Use it in workflows to automate social media operations with credentials passed as action inputs.
